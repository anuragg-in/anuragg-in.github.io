Testing of digital circuits using Beaglebone Black

The aim of this project was to devise a python based software on Beaglebone black to debug and validate mock-up circuit design on breadboard.

Strategy

  • Accept description of the device under test.
  • Scan through all possible sets of input for combinational circuit and minimal set of states and transition using postman algorithm for sequential circuit.
  • Displaying the result on a graphical user interface (GUI).
